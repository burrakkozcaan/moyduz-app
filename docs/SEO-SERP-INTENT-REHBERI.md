# SERP Analizi & Arama Niyeti (Search Intent) Rehberi

Yeni sayfa açmadan veya içerik üretmeden önce: **SERP’e bak → Niyeti belirle → Aynı formatta sayfa üret.**

---

## 1. SERP’te Ne Var?

Hedef kelimeyi Google’da arat ve ilk 3–5 sonucun **türünü** not et:

| Sonuç türü | Google’ın mesajı |
|-------------|------------------|
| **Kategori / ürün sayfası** | Kullanıcı çözüm/hizmet arıyor → Landing / hizmet sayfası |
| **Blog / rehber** | Bilgi arıyor → Informational içerik |
| **Landing page** | Karar aşaması / fiyat bilgisi → Commercial/Transactional landing |
| **Liste (“En iyi 10 …”)** | Karşılaştırma istiyor → Comparison/list sayfası |
| **Fiyat sayfası** | Fiyat niyeti güçlü → Pricing sayfası veya landing’de fiyat bloğu |
| **FAQ / SSS** | Soru-cevap niyeti → FAQ bloğu + FAQPage schema |

**Kural:** SERP’te hangi format ağır basıyorsa, sen de o formatta sayfa üretmelisin.

---

## 2. Dört Niyet Tipi

| Intent | Örnek sorgular | Doğru sayfa formatı |
|--------|----------------|----------------------|
| **Informational** | “nedir”, “nasıl yapılır”, “ne demek” | Blog, rehber, SSS |
| **Commercial** | “en iyi”, “fiyat”, “karşılaştırma”, “alternatif” | Landing (özellik + fiyat), compare sayfası, list içerik |
| **Transactional** | “satın al”, “demo al”, “kurulum”, “teklif” | Landing + CTA, contact, pricing |
| **Navigational** | Marka / ürün adı | Ana sayfa, hizmet sayfası, brand sayfaları |

## 4. Yeni İçerik Öncesi Kontrol Listesi

1. **SERP’i aç** → Hedef kelimeyi Google’da ara.
2. **İlk 3 sonucun türünü yaz** → Blog mu, landing mi, liste mi, fiyat mı?
3. **Niyeti kararlaştır** → Informational / Commercial / Transactional / Navigational.
4. **Formata karar ver** → Aynı niyet için doğru sayfa tipi (blog vs landing vs compare).
5. **Mevcut sayfaya bağla** → Aynı keyword için zaten sayfa varsa güçlendir; yoksa yeni sayfa planı yap.

---

## 5. Kısa Özet

- **SERP = kullanıcı niyetinin göstergesi.** Aynı formatta içerik üretmek sıralama şansını artırır.
- **“pazaryeri yazılımı”** → Çoğunlukla landing / ürün sayfası çıkıyorsa → Sen de landing (✅ `/cok-saticili-e-ticaret-altyapisi`).
- **“pazaryeri yazılımı nedir”** → Blog / rehber çıkıyorsa → Sen de blog (✅ marketplace-nedir, multi-vendor-nedir).
- **“pazaryeri yazılımı fiyat”** → Fiyat sayfası veya landing’de fiyat çıkıyorsa → Fiyat bloğu + CTA (✅ pricing + CTA).

Bu rehberi yeni keyword eklerken veya SERP’e göre sayfa tipi seçerken kullan.
