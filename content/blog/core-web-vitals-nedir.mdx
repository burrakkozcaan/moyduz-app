## Core Web Vitals Nedir?

Core Web Vitals, Google'ın web sayfalarının kullanıcı deneyimini ölçmek için kullandığı üç ana performans metriğidir. 2021'den itibaren doğrudan SEO sıralama faktörü olarak kullanılıyor.

## Üç Ana Metrik

### 1. LCP — Largest Contentful Paint

Sayfanın en büyük içerik öğesinin (genellikle hero görseli veya başlık) yüklenmesi için geçen süre.

- **İyi:** < 2.5 saniye
- **Geliştirilmeli:** 2.5-4.0 saniye
- **Kötü:** > 4.0 saniye

**E-ticarette LCP sorunlarının başlıca nedeni:** Optimize edilmemiş ürün görselleri.

**Çözüm:**
- WebP formatı
- Görsel sıkıştırma (TinyPNG, Squoosh)
- CDN kullanımı
- Lazy loading

### 2. CLS — Cumulative Layout Shift

Sayfa yükleme sırasında öğelerin beklenmedik şekilde kayması.

- **İyi:** < 0.1
- **Geliştirilmeli:** 0.1-0.25
- **Kötü:** > 0.25

**E-ticarette CLS sorunu:** Yüklenen reklam bannerları sayfayı kaydırdığında kullanıcı yanlış butona tıklıyor.

**Çözüm:**
- Görseller için boyut tanımlama (width/height attribute)
- Yazı tiplerini önceden yükleyin
- Dinamik içerik için yer ayırın

### 3. INP — Interaction to Next Paint

Kullanıcı etkileşimlerine (tıklama, dokunma) sitenin ne kadar hızlı yanıt verdiği.

- **İyi:** < 200ms
- **Geliştirilmeli:** 200-500ms
- **Kötü:** > 500ms

## Core Web Vitals'ı Nasıl Ölçersiniz?

**Araçlar:**
- Google PageSpeed Insights (ücretsiz)
- Google Search Console (gerçek kullanıcı verisi)
- Lighthouse (Chrome DevTools)
- GTmetrix

## E-Ticaret ve Core Web Vitals

E-ticaret sitelerinde CWV genellikle düşük çünkü:
- Çok sayıda yüksek çözünürlüklü ürün görseli
- Ağır JavaScript (eklentiler, chat, analytics)
- Yetersiz optimize hosting
