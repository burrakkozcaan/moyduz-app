## Hızlı Cevap

**API entegrasyonu**, e-ticaret sisteminizdeki ödeme, kargo, ERP ve pazaryeri bağlantılarını otomatize eder. Entegrasyonsuz mağaza Trendyol/Hepsiburada siparişlerini manuel girmek zorunda kalır — **günde 50+ siparişte bu sürdürülemez**. Minimum gerekli entegrasyonlar: ödeme (iyzico/PayTR), kargo (Yurtiçi/Aras), pazaryeri API'si.


## API Nedir?

API (Application Programming Interface — Uygulama Programlama Arayüzü), farklı yazılım sistemlerinin birbiriyle veri alışverişi yapmasını sağlayan standart bir iletişim protokolüdür.

Basit bir analoji: API bir restoranın garsonuna benzer. Mutfağa (backend) ne istediğinizi söyler, yanıtı size getirir. Mutfağın nasıl çalıştığını bilmenize gerek yok.

## E-Ticarette API Entegrasyonları

### Ödeme API'leri

iyzico, PayTR, Stripe gibi ödeme sistemleri API üzerinden entegre edilir. Müşteri kart bilgisini girer → e-ticaret sitesi → ödeme API → banka → onay/red → e-ticaret sitesi.

[Özel e-ticaret sistemleri](/ozel-e-ticaret) tüm Türk ödeme sistemleriyle sorunsuz API entegrasyonu sunar.

### Kargo API'leri

Sipariş oluşturulduğunda kargo firmasına otomatik bildirim gönderilir, kargo barkodu üretilir, takip numarası müşteriye iletilir. Tüm bunlar API ile saniyeler içinde gerçekleşir.

### ERP API'leri

Sipariş geldiğinde stok otomatik düşer, fatura kesilir, muhasebe kaydı oluşur. Logo, Mikro, SAP gibi ERP'lerle API entegrasyonu operasyonel verimliliği artırır.

[B2B e-ticaret](/b2b-ecommerce) sistemlerinde ERP entegrasyonu kritik öneme sahiptir.

### Pazaryeri API'leri

Trendyol, Hepsiburada, Amazon gibi pazaryerleri API'leri üzerinden ürün yönetimi ve sipariş takibi yapılabilir.

## API Türleri

**REST API:** En yaygın kullanılan format. HTTP protokolüne dayalı.

**GraphQL:** Daha esnek veri sorgulama. Facebook tarafından geliştirilen.

**Webhook:** Belirli bir olay gerçekleşince otomatik bildirim gönderme.

## API Entegrasyon Süre ve Karmaşıklık Tablosu

| Entegrasyon Türü | Süre | Zorluk |
|---|---|---|
| Basit ödeme (iyzico/PayTR) | 1–3 gün | Düşük |
| Kargo API (Yurtiçi/Aras) | 3–7 gün | Düşük |
| Pazaryeri (Trendyol/Hepsiburada) | 1–2 hafta | Orta |
| ERP (Logo/Mikro, tek yönlü) | 1–2 hafta | Orta |
| ERP (çift yönlü, gerçek zamanlı) | 2–6 hafta | Yüksek |

## Karar Matrisi: Ne Zaman Hangi Entegrasyon?

| Durum | Öncelikli Entegrasyon |
|---|---|
| Yeni açılan mağaza | Ödeme + kargo (zorunlu minimum) |
| Günde 30–50 sipariş | Pazaryeri API ekle |
| Günde 100+ sipariş | ERP stok senkronizasyonu zorunlu |
| B2B kanal açma | Özel fiyatlandırma + ERP entegrasyonu |
