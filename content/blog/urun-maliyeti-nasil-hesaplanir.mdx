## Gerçek Ürün Maliyeti Nedir?

Çoğu e-ticaret satıcısı yalnızca alış fiyatını maliyet zanneder. Bu büyük bir hatadır.

**Gerçek ürün maliyeti = Alış + Tüm Gizli Maliyetler**

## Maliyet Bileşenleri

### 1. Direkt Maliyetler

| Kalem | Açıklama |
|-------|----------|
| Alış fiyatı | Tedarikçiden ödenen fiyat |
| İthalat gümrüğü | Yurt dışı ürün alımında |
| Kargo (gelen) | Tedarikçiden depoya |
| Paketleme malzemesi | Kutu, bant, bubble wrap |

### 2. Gizli Operasyonel Maliyetler

| Kalem | Tipik Tutar |
|-------|------------|
| Depolama (aylık/ürün) | 1-5 ₺ |
| İşçilik (hazırlama) | 2-10 ₺/sipariş |
| Kargo (giden) | 15-35 ₺ |
| İade işleme | 10-25 ₺/iade |

### 3. Platform/Komisyon Maliyetleri

| Kalem | Tutar |
|-------|-------|
| Marketplace komisyonu | %8-18 |
| Ödeme işlemcisi | %1.5-3 |
| Platform aboneliği | Aylık sabit ücret |

### 4. Pazarlama Payı

Ürün bazında reklam maliyeti bölünmeli.

Aylık 10,000 ₺ reklam → 500 sipariş → ürün başı 20 ₺ pazarlama payı.

## Formül

```
Toplam Maliyet = Alış + Kargo (giden) + Paketleme
              + Komisyon + Ödeme ücreti + Pazarlama payı
              + İade payı + Genel gider payı
```

## Pratik Örnek

**Ürün:** Güneş gözlüğü, satış fiyatı 300 ₺

| Kalem | Tutar |
|-------|-------|
| Alış fiyatı | 60 ₺ |
| Paketleme | 5 ₺ |
| Kargo (giden) | 18 ₺ |
| Marketplace komisyonu (%12) | 36 ₺ |
| Ödeme ücreti (%2) | 6 ₺ |
| Reklam payı | 25 ₺ |
| İade payı (%5 oranında) | 5 ₺ |
| **Toplam maliyet** | **155 ₺** |
| **Net kar** | **145 ₺ (%48)** |

Fena görünüyor değil mi? Ama:
- KDV sonrası: 300 ₺ → KDV %20 dahil → 250 ₺ net satış
- Yeniden hesap: net kar 95 ₺ (%38)

Bu yüzden KDV dahil mi hariç mi? sorusu kritik.

## Satış Fiyatı Belirleme

**Hedef kar marjına göre:**
```
Satış Fiyatı = Toplam Maliyet / (1 - Hedef Kar Marjı)
```

**Örnek:**
- Toplam maliyet: 80 ₺
- Hedef net kar marjı: %30
- Satış fiyatı: 80 / (1 - 0.30) = **114 ₺**

## Fiyatlandırma Stratejileri

**Maliyet+:** Maliyetin üzerine sabit oran ekle. Basit ama rakipleri görmezden gelir.

**Rekabetçi:** Rakip fiyatların altında/üstünde/aynı konumlanma.

**Değer bazlı:** Müşterinin ürüne atfettiği değer. Niş ürünlerde en yüksek kar marjı.

## Komisyonu Azaltmanın Etkisi

100 ₺ ürün üzerinde %12 komisyon = 12 ₺. Kendi [özel e-ticaret sistemine](/ozel-e-ticaret) geçişte bu sıfırlanır.

Aylık 500 sipariş × 12 ₺ tasarruf = 6,000 ₺/ay = **72,000 ₺/yıl tasarruf**.
