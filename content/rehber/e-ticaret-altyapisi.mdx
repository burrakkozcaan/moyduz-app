## Rakip Platformların Gizli Maliyetleri

SaaS platformlarda fiyat listesindeki rakam, gerçekte ödeyeceğiniz rakam değildir:

**Ek modül ücretleri:** T-Soft, Ticimax ve IdeaSoft'ta B2B, multi-vendor veya gelişmiş raporlama için ek modül satın almanız gerekir. Bu ücretler temel pakete yakın seviyelere ulaşabilir.

**ikas kargo bakiyesi tuzağı:** ikas'ın sunduğu "ücretsiz kargo bakiyesi", aslında müşteriyi hızla platforma kilitleyen bir stratejidir. Kargo entegrasyonu aktifleştikten sonra platform bağımlılığı hızla artar.

**AI içerik araçları gerçekte ne yapıyor?** Ticimax AI, bulk ürün açıklaması üretimi ve meta tag optimizasyonu sunar. Bu içerik hacmi ve hız sağlar — ama "bu kategori %18 dönüşüm kaybediyor" gibi veri odaklı SEO önerileri sunmaz. Hız ≠ sonuç.

## 1) Altyapı Türleri: Hangisi Sana Uygun?

| Tür | Avantaj | Dezavantaj | Kimler için |
|---|---|---|---|
| SaaS (Shopify, ikas) | Hızlı kurulum | Özelleştirme sınırlı | Başlangıç |
| Headless + Custom | Tam kontrol | Daha uzun geliştirme | Büyüyen markalar |
| Marketplace (Trendyol) | Hazır trafik | Komisyon + bağımlılık | Ek kanal |

**Moyduz yaklaşımı:** Özel Next.js altyapısı — hem SEO hem hız hem ölçek.

---

## 2) Teknik Stack Seçimi

### Frontend
- **Next.js 14+** — SSR + SSG + Edge ready
- **Tailwind CSS** — hızlı UI
- **TypeScript** — güvenli kod tabanı

### Backend / API
- Headless CMS (Sanity, Contentful)
- REST veya GraphQL API
- Webhook tabanlı stok/sipariş

### Ödeme
- iyzico, PayTR, Shopier — TR için yaygın
- Stripe — uluslararası

### Hosting
- Vercel veya Cloudflare Workers — edge + CDN
- TTFB < 200ms hedefi

---

## 3) Lojistik & Operasyon Entegrasyonu

1. **Kargo anlaşması** — MNG, Yurtiçi, Aras, Sendeo
2. **Stok yönetimi** — Entegrasyon veya manuel API
3. **İade süreci** — otomatik etiket + geri ödeme
4. **E-fatura** — GİB entegrasyonu (zorunlu >30K TL ciro)

---

## 4) Teknik SEO Altyapısı

- Sayfa hızı: Core Web Vitals (LCP < 2.5s, INP < 200ms)
- Yapısal veri: Product + BreadcrumbList + FAQPage schema
- Canonical + hreflang (çok dil için)
- Sitemap + robots.txt otomasyonu
- IndexNow entegrasyonu — anlık index
